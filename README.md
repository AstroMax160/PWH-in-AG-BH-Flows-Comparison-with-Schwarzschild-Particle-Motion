# Maximo Prediger - BEP work on 2D PWH + AG


## Report Abstract - (Access to the report may be restricted, ask project supervisor)

A millimetric droplet bouncing on the surface of a vertically vibrating liquid bath can become self-propelled by interacting with the waves generated by its own previous impacts. The resulting coupled droplet-wave system, known as a walker, forms the basis of Pilot-Wave Hydrodynamics and reproduces several particle-like and quantum-like behaviours. Separately, surface waves propagating against a moving fluid can experience a blocking boundary analogous to a black-hole horizon. This project investigated whether these two systems can be combined to create a two-dimensional single-particle analogue of Schwarzschild motion. A partial differential equation model for a walking droplet was extended by introducing a prescribed inward radial background flow. The surface-wave equations were modified to include advection, while the relevant term in the droplet impact dynamics was written relative to the local bath velocity. Requirements were also derived to ensure that the background flow remained compatible with the Faraday-wave response and sufficiently stable in the region used to analyse the walker.

The resulting walker trajectories were compared with nondimensional Schwarzschild geodesics. For radial infall from rest, the walker crossed the horizon with a speed within approximately 6.1% of the general-relativistic reference, although the crossing occurred about 27% earlier. Trapping tests showed that neither the phase nor group wave horizon formed a strict no-escape boundary for the walker: droplets starting sufficiently far inside remained trapped, whereas droplets initialised closer to the horizon could escape. The radial configuration also failed to reproduce circular Schwarzschild motion because the azimuthal droplet velocity was damped without a mechanism to sustain it. The inward radial system therefore retains a Schwarzschild-like wave structure, but does not provide a general correspondence between walking-droplet and Schwarzschild particle trajectories. As a separate exploratory extension, an azimuthal component was added to create a draining-vortex background flow corresponding to a Kerr-like analogue-gravity configuration. This rotating flow sustained the walker’s azimuthal motion more effectively and produced a substantially more rotational trajectory. Although no comparison with Kerr geodesics was performed, the result indicates that the Kerr-like configuration provides a promising direction for further development.

## Project Note

While this project worked on strictly radial flow, as detailed in the report, the code used can easily be adapted to radial+rotational background flow by adding the necessary terms to vx_B and vy_B in lines 273 and 274 of PWH_AG_CPU_v1.py. That may have consequences for the flow requirements and analysis performed in the report, but it can be useful as a future project based on the code. There is a version of the code with background flow already implemented, marked by "azimuthal" at the end.

## Important compatibility notes

For the codes, they were done for Python 3.10, Numpy 2.2.6, and Cupy 14.1.1 in a CUDA 12.2 environment.
Cupy can be especially sensitive to Cupy and CUDA version, so I cannot ensure these codes work beyond the versions used for this project.

Furthermore, it is recommended to keep the folders as they are organised which ensures the codes work properly. If downloaded and reorganised, make sure to also review the different running or visualising codes to update the paths.
